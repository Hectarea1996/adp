# Style-maker interface

## Style parameters

#### ***Macro*** ADP-PRIVATE::DEF-STYLE-PARAMETER

```Lisp
(DEFMACRO ADP-PRIVATE::DEF-STYLE-PARAMETER
          (ADP-PRIVATE::NAME
           &KEY (ADP-PRIVATE::VALUE NIL) (ADP-PRIVATE::KEY-NAME NIL)
           (ADP-PRIVATE::REQUIRED NIL))
  (CHECK-TYPE ADP-PRIVATE::NAME SYMBOL "a symbol")
  (CHECK-TYPE ADP-PRIVATE::KEY-NAME KEYWORD "a keyword")
  `(PROGN
    (DEFPARAMETER ,ADP-PRIVATE::NAME ,ADP-PRIVATE::VALUE)
    (IF ,ADP-PRIVATE::KEY-NAME
        (ADP-PRIVATE::ADD-STYLE-PARAMETER ,ADP-PRIVATE::NAME
                                          ,ADP-PRIVATE::KEY-NAME
                                          ,ADP-PRIVATE::REQUIRED)
        (ADP-PRIVATE::ADD-STYLE-PARAMETER ,ADP-PRIVATE::NAME
                                          (INTERN
                                           (SYMBOL-NAME ,ADP-PRIVATE::NAME)
                                           :KEYWORD)
                                          ADP-PRIVATE::*STYLE-PARAMETERS*))))
```

## Customizable writer macros

#### ***Macro*** ADP-PRIVATE:DEF-HEADER-WRITER

```Lisp
(DEFMACRO ADP-PRIVATE:DEF-HEADER-WRITER (#:WRITER-ARGS1 &BODY #:BODY-ARG2)
  (CHECK-TYPE #:WRITER-ARGS1 LIST)
  (LOOP ADP-PRIVATE::FOR #:WRITER-ARG0 ADP-PRIVATE::IN #:WRITER-ARGS1
        DO (CHECK-TYPE #:WRITER-ARG0 SYMBOL))
  `(SETQ ,'ADP-PRIVATE::*HEADER-PROC* (LAMBDA ,#:WRITER-ARGS1 ,@#:BODY-ARG2)))
```
