#### Macro: DEF-STYLE-PARAMETER

```Lisp
(defmacro ADP-PRIVATE::DEF-STYLE-PARAMETER (NAME &KEY (VALUE NIL)
                                            (KEY-NAME NIL) (REQUIRED NIL))
  ...)
```
